**Aura** is an intelligent stock investment and market analysis application built with Flutter. Serving as a graduation project, it leverages Artificial Intelligence to provide users with smart recommendations, market sentiment analysis, and portfolio management tools.

*(Note: This project is currently under active development)*

---

## ✨ Key Features

* **AI-Powered Insights:** Smart stock recommendations and AI signals based on market trends and technical analysis.
* **Market Analysis & Sentiment:** Track stock details, sentiment gauges, and key statistics.
* **Smart Automation:** Setup automated portfolio actions, order values, and real-time alerts.
* **Stock Comparison:** Compare multiple stocks and view detailed ratio analyses to make informed decisions.
* **Bilingual Support:** Full localization for both English and Arabic.
* **Secure Authentication:** Standard login alongside biometric (Face ID) authentication.
